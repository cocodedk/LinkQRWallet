# Link QR Wallet

Link QR Wallet is a personal Android app that turns URLs into QR codes and stores them in a searchable, sortable library. Save links fast, scan existing QR codes, and keep everything offline-first.

## Features
- Add links, auto-fetch page titles, and generate QR codes
- Search and sort by title, domain, or date
- QR scanner to import existing codes
- Share QR images and open links quickly
- Local database (offline-first)

## Build
```bash
./gradlew assembleDebug
```
